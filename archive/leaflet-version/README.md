# Leaflet Version (Archive)

This is the original Leaflet + OpenStreetMap implementation of Hyakumeiten Map.

To use this version, copy the files from this directory to the parent directory and reload.

Features:
- Leaflet map with OpenStreetMap tiles
- MarkerClusterer for grouped markers
- CSV parsing with PapaParse
- Collapsible filter UI (region, genre, price)
- Geolocation with radius filtering
- Lazy-loaded images in popups
